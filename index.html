Conecte-se na CodeSquad

A CodeSquad é um grupo de amigos que se juntou
para estudar programação e criar projetos
juntos. Compartilhamos recursos e
conhecimentos para ajudar uns aos outros a
desenvolver habilidades de programação e a criar
projetos criativos e desafiadores.

Objetivos

O objetivo da CodeSquad é se tornar uma equipe de desenvolvimento de software de sucesso, enquanto aprendem juntos e se divertem.

Nossa Comundade

O grupo se reúne regularmente para discutir tópicos relacionados à programação, como linguagens de programação, frameworks e ferramentas.

Ferramentas

A CodeSquad utiliza diversas ferramentas e plataformas para auxiliar nos estudos, como o GitHub para controle de versão e colaboração em código
